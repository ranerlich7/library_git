fork this project.
1. clone this project to a folder:
https://github.com/ranerlich7/library_git.git
2. install virtual env
python -m venv venv
3. activate virtual env
.\venv\Scripts\activate
4. install packages
pip install -r requirements.txt
5. run
python book.py
